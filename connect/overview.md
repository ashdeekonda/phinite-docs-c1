Phinite **Connect** covers two related layers:

| Layer | What it does | Where to go |
| ----- | ------------ | ----------- |
| **Channels** | Where assistants **talk to users** — webchat, messaging, voice, email | [Channels setup guide](/channels/guide) |
| **Integrations Hub** | Prebuilt **tools** agents call — CRM, email, databases, DevOps, and more | [Integrations Hub overview](/integrations-hub/overview) |

## When to use channels

Configure a channel when you deploy a conversational or voice assistant and need users to reach it outside Graph Studio — embedded webchat, WhatsApp, Slack/Teams, Twilio, or email inboxes.

See the [Channels setup guide](/channels/guide) for per-channel configuration.

## When to use Integrations Hub

Configure Integrations Hub when agent graphs or DevStudio tools need to read/write external systems (Gmail, Salesforce, GitHub, databases, etc.). Builders attach integrations in Graph Studio or DevStudio; credential setup may use the public agent-config flow for external callers.

Browse the full catalog: [Integrations Hub overview](/integrations-hub/overview).
