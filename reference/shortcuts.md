## Flowgen Studio

- ⌘/Ctrl + Click: Multi-select
- Delete/Backspace: Delete selected edge/node
- Space + Drag: Pan canvas
- ⌘/Ctrl + + / -: Zoom in/out

## DevStudio

- ⌘/Ctrl + Enter: Run tool test
- ⌘/Ctrl + S: Save
