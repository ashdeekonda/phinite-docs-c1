## Methods

Phinite offers two ways to build agent graphs in **Graph Studio**:

- **[Phinite Aura method](/graph-studio/copilot-method)** — Describe what you want in natural language; Aura drafts and updates the graph on the canvas.
- **[Manual method](/graph-studio/manual-method)** — Add and connect nodes yourself using the node library and inspector.

## Choose a method

- Use **Phinite Aura** to bootstrap quickly or iterate on structure with prompts.
- Use **Manual** for precise control, complex branching, or when you already know the graph layout.

Both methods work on the same agent graph. You can switch between Aura and manual editing at any time.
