## Timeline view

Step-by-step execution trace of agent graphs with timestamps and outcomes.

**Use cases:** Identify slow steps and bottlenecks, locate failures and exceptions, verify conditional routing paths

## Decision joints and exceptions

Inspect branching decisions and exception paths during execution.

## Variable capture logs

Review captured session variables at each step for debugging data flow.

## Best practices

- Filter by agent graph version and environment
- Trace failures to specific tool calls
- Export logs for audits
