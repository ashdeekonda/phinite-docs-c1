# MCP Server

**MCP Server** in the workspace sidebar lets you register and manage **Model Context Protocol (MCP)** servers that tools and agent graphs can call.

## What it is for

- Connect external MCP-compatible services to Phinite
- Reuse MCP tools when exposing agents (see **Additional Config → MCP** in the [Expose wizard](/agent-registry/expose-your-flow))
- Centralise server configuration at workspace scope

## Access

Requires the `workspace.sidebar.mcp_servers` permission. All roles that can open workspace **Tools** and **Integrations** typically have access unless your organisation restricts it.
