## Methods

- **Phinite Aura** — Describe functionality; Aura drafts a tool in Dev Studio
- **Manual coding (Python)** — Implement handlers and schemas yourself

## Choose a method

- Use **Phinite Aura** to bootstrap quickly
- Use manual coding for complex or sensitive logic
