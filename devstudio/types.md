## Types you build in DevStudio

| Type | Description |
| --- | --- |
| **Custom tools** | Purpose-built logic for your business — code or Phinite Aura |
| **System tools** | Platform-provided capabilities |
| **Structure tools** | Typed input/output schemas shared across tools |

## Predefined tools (Integrations)

**Predefined tools** are not built in DevStudio. They are pre-packaged third-party integrations (Gmail, Slack, Jira, Salesforce, MongoDB, and [80+ more](/integrations-hub/overview)) configured under **Integrations → Predefined tools**.

Once connected, they appear in Graph Studio's tool picker like any other tool.
